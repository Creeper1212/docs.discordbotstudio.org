Multiple Bots

To create a new bot, or start working on another bot, go to the choose bot tab. From here, you can select an existing bot from the dropdown to work on, or create a new bot.
First you will need to choose a folder to place the new bot files in. To do this, click the choose folder button, and select a folder. Next, give the bot a name, and click save bot. After saving, you will be able to select the new bot you created from the dropdown.
