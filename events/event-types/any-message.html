Any Message

How to use the Any Message event in Discord Bot Studio
This event is emitted whenever a message is created.
tempVars.anymessage is the variable containing the message that was sent to trigger this event. tempVars.anymessagemember is the variable containing the member who sent this message to trigger the event. Both can be leveraged in response to the event.

Example response

Connecting the above response to the Any Message event will send a message tagging the user who sent a message and showing the name of the channel they sent that message in.
