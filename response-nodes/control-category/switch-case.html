Switch Case

Functions like a switch case node in traditional programming. Allows a variable to be compared to any amount of values.
The output for which the variable value is equal will be executed. If no value matches, none of the outputs will be executed.
Field
Description
Variable to check
The variable whose value will be compared with possible values.
Possible values
Possible values the variable could match. Each possible value will be added as an output to the node. If the value matches, the responses connected to that output will be executed.
