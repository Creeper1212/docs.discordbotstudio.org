Check if Variable Exists

A multiple-output node that checks whether a variable exists.
If the variable supplied exists, the nodes connected to the true output will be run, otherwise the nodes connected to the false output will be run.
Field
Description
Name of variable to check
The name of the variable whose existence is being checked. Do not include the ${} syntax. For example- if the variable name is ${tempVars.myValue}, you would just enter myValue in the field.
What type of variable is this
Select what type the variable that is being checked is stored as.
