Check Variable Value

A multiple-output node that allows you to check the value of a certain variable.
If the condition you supply is true, the nodes connected to the true output will be run, otherwise the nodes connected to the false output will be run.
Field
Description
Name of variable to check
The name of the variable whose value is being checked. Do not include the ${} syntax. For example- if the variable name is ${tempVars.myValue}, you would just enter myValue in the field.
What type of variable is this
Select what type the variable that is being checked is stored as.
If variable is
Pick an operator for comparison.
Value
The value the variable will be compared to.
