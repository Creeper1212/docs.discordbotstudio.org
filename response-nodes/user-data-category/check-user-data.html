Check User Data

A multiple-output node that checks the value of a user data field.
This node allows the value of a user data field to be checked. If the condition provided evaluates to true the nodes connected to the true output will be run, otherwise the nodes connected to the false output will be run.
Field
Description
User to check data from
The user whose user data field will be checked. This should evaluate to an id; you can manually copy a user's id or get it programmatically by using a user type variable.
Data field
The name of the field whose value will be checked.
If data field is
The operation to be used to compare values.
Value
The value to compare with the user data field.
