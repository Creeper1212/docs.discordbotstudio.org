Check if Message is in Channel

Checks whether or not a message is in a certain channel. Evaluates to true if it is in the channel, and false otherwise.
​
Description
Message
The id of the message to be checked. You can use the "Insert Variable" button to insert a variable that will evaluate to a message id.
Channel
The name or id of the channel that should be checked.
