Send Image

Sends an image to a specified channel
Field
Description
Channel
Put the name or id of a channel to send the message to here. You can use the "Insert Variable" button to insert a variable that will evaluate to a channel name or id.
URL
The image URL. Make sure this ends in something like "jpg" or "png" so it shows the image and doesn't attach it as a file.
